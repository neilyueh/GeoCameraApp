# 📸 GeoCamera - 地理位置相機

**GeoCamera** 是一個簡單好用的相機 App，專為需要紀錄「時間」與「地點」的使用者設計。當您按下快門時，App 會自動將**日期**、**時間**、**GPS 座標**以及**詳細地址**壓印在照片上，讓您的每一張照片都包含完整的地理資訊。

非常適合：

* 🏗️ **工程紀錄**：工地施工狀況回報。
* 🏃 **外勤打卡**：業務巡點、外送紀錄。
* ✈️ **旅遊記憶**：紀錄旅行的足跡與地點。

---

## ✨ 主要功能

* **📍 即時定位**：畫面右下角即時顯示當前的經緯度與地址。
* **🖼️ 自動浮水印**：拍照後，資訊會直接「燒」在照片上（右下角），無法修改，確保真實性。
* **📅 時間戳記**：精確紀錄拍照的年、月、日與時間。
* **🔄 智慧轉向**：支援**直立**與**橫向**拍攝，浮水印會根據手機方向自動調整位置與角度。
* **💾 自動存檔**：拍攝完成的照片會自動儲存到您的系統相簿中。

---

## 🚀 如何開始

1. 開啟 App。
2. **授權請求**：第一次使用時，請允許以下權限：
    * 📷 **相機**：用於拍攝畫面。
    * 📍 **位置**：用於取得 GPS 與地址資訊。
    * 🖼️ **相簿**：用於儲存完成的照片。
3. 等待畫面右下角的地址資訊出現（顯示為白色文字）。
4. 按下下方的 **圓形快門鈕** 拍照。
5. 聽到「喀擦」聲並感受到震動後，照片即儲存成功！
